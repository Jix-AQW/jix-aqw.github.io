# Limited Quantity Sets (LQS)
---

!!! info "O que é LQS?"
    *Limited Quantity Shops* (LQS) são pacotes de itens **raros e limitados** no AdventureQuest Worlds.
    Cada set possui uma **quantidade fixa global** e, quando esgota, **não volta mais**.

Os LQS existem principalmente para **colecionadores**. São alguns dos itens mais valorizados do jogo justamente por sua **escassez controlada**.

## O que são Limited Quantity Shops?

Os Limited Quantity Shops funcionam de forma simples:

- Cada pacote possui um **número máximo de unidades**
- Esse número é o **mesmo para todos os jogadores, no mundo inteiro**
- Quando o estoque chega a zero, o set é considerado **esgotado permanentemente**

Exemplo:  
Se a página do LQS mostra *300 itens restantes*, isso significa que **existem apenas 300 unidades disponíveis no total**, não por servidor ou região.

---

## Como funcionam os LQS?

O processo de compra e resgate acontece **fora do jogo** e depois **dentro do AQW**:

1. Acesse o **Account Manager** no site oficial da AQ  
2. Entre na seção **Limited Quantity Shop**  
3. Compre o set usando **AdventureCoins (ACs)** ou **Gold**, dependendo do pacote  
4. Entre no AQW  
5. Abra o **Book of Lore**  
6. Acesse o **badge do set comprado**  
7. Resgate todos os itens por **0 ACs** (free storage)

!!! tip "Dica"
    Todos os itens de LQS custam 0 ACs no badge, então podem ficar no banco **para sempre**, sem custo.

---

## Restocks: como funcionam?

Para dar mais chances a jogadores de diferentes fusos horários:

- O estoque total **não é liberado de uma vez**
- Apenas uma parte é disponibilizada inicialmente
- Novas unidades são adicionadas em **restocks ao longo do fim de semana**

Isso melhora as chances de conseguir um item, mas **não garante** que todos conseguirão.

!!! warning "Importante"
    Mesmo com restocks, **não há itens suficientes para todos**.
    Quanto menor a quantidade total, mais raro o set.

---

## Onde encontrar os Limited Quantity Sets?

Os links para compra dos LQS podem ser encontrados em:

- Design Notes oficiais
- Página inicial do site
- Upgrade Page
- NPC **Khuddar Khamundi**, o Black Market Vendor, em Battleon  
  (ele fornece um link direto para a página do LQS)

---

## As quantidades atualizam automaticamente?

Não.

- As quantidades **só atualizam** quando você:
  - Recarrega a página
  - Compra um item
- Se aparecerem poucas unidades restantes, **aja rápido**
- Um set pode esgotar **entre um clique e outro**

---

## Requisitos importantes

- Todos os itens de LQS são **0 ACs no badge**
- Sets de **Legend/Member** podem ser comprados mesmo sem upgrade ativo  
  desde que sua conta **já tenha sido upgrade em algum momento no passado**

---

## História e contexto dos LQS no AQW

Desde os primeiros anos do AdventureQuest Worlds, itens raros sempre tiveram um papel especial na comunidade.
Inicialmente, muitos rares surgiam de eventos ou mapas temporários, o que tornava difícil controlar sua exclusividade.

Os **Limited Quantity Shops** surgiram como uma solução mais justa e transparente:
em vez de depender apenas do tempo online ou de eventos únicos, os jogadores passaram a disputar itens com base em **quantidade limitada global**.

Isso transformou os LQS em símbolos de status, memória e história dentro do jogo.
Cada set representa não apenas um visual exclusivo, mas também um momento específico da linha do tempo do AQW.

Para muitos jogadores, possuir um LQS é como carregar um pedaço da história de Lore.

---

## Considerações finais

Os Limited Quantity Sets não são feitos para todos, e essa é exatamente a ideia.
Eles existem para valorizar a raridade, a coleção e o registro histórico do jogo.

Se você busca itens exclusivos e gosta da ideia de colecionar raridades,
os LQS são um dos sistemas mais importantes do AdventureQuest Worlds.
